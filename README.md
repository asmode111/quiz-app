# Running APP

An experimental project for a quiz app.

Built with [Node](https://nodejs.org) stack.
Handling the frontend with [React](https://reactjs.org).

### Tech stack

- Nodejs

- ReactJS

- Typescript

- SQL Lite

### Run the project

```
./quiz.sh build
./quiz.sh start
```

Run API

http://localhost:8081/

Run APP

http://localhost:8080/
